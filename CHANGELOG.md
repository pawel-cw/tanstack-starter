# Changelog

All notable changes to this boilerplate will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-10-31

### Added
- Initial release of TanStack Starter boilerplate
- TanStack Start full-stack framework setup
- Better-Auth authentication system with email/password
- Neon Postgres database with Drizzle ORM
- shadcn/ui component library with Tailwind CSS v4
- Lingui internationalization (English, German, Polish)
- TanStack Router with type-safe routing
- TanStack Query for server state management
- Complete development environment with TypeScript
- ESLint and Prettier configuration
- Comprehensive README and QUICKSTART guide
- Example homepage showcasing tech stack
- Database migrations setup with Drizzle Kit
- Environment variable configuration with .env.example
- Development tools and scripts
- TanStack DevTools integration

### Infrastructure
- Vite build configuration
- Path aliases for clean imports
- Dark mode support
- Responsive design utilities
- Production-ready build setup
